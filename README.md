# RATEM

**Regime-Aware Time-to-Event Modeling Pipeline**

A lightweight framework for building **auditable, reproducible, and regime-aware** time-to-event workflows.

## What RATEM is
RATEM is a methodological framework for trustworthy survival / time-to-event analysis.

It emphasizes:

- explicit assumptions
- data contracts
- missingness transparency
- reproducible preprocessing
- artifact-backed analysis
- regime-aware modeling decisions

## RATEM Field Notes
This repository includes **RATEM Field Notes**, a public build-in-public series documenting the framework one step at a time.

Core principle:

**Assumptions → Evidence → Artifacts**
