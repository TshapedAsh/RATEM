# Week 4 Concept — Hazard vs Probability

Week 4 separates three related but non-interchangeable quantities:

- **S(t)** — survival probability
- **F(t) = 1 - S(t)** — event probability by time *t*
- **h(t)** — hazard, a local conditional event intensity

The main methodological warning is simple:

**hazard is not itself a probability**

That distinction matters before subgroup Kaplan–Meier curves and later Cox-model interpretation.
